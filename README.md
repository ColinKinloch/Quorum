Quorum
======

It's for paries, yo!

Installation
------------

1. install nodejs, npm and ruby
2. run "npm install -g bower" (optional?)
3. run a mysql server
4. run "npm install"
5. run "bower install"
6. run "sudo gem install compass"

Running
-------

1. run "grunt serve --force"
2. open a browser at "localhost:9000" (not 127.0.0.1:9000)
